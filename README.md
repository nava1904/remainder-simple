



# Remainder Simple

A simple task/reminder app built for iPhone using Swift and Xcode, for personal use.

---

## 🚀 Overview

**Remainder Simple** is a lightweight iOS app to help you manage reminders or tasks directly on your phone. The goal is minimalism — quick to use, no extra fluff, just tasks and reminders.

---

## ✨ Features

- Add, view, delete reminders/tasks  
- Simple UI/UX that’s easy to navigate  
- Local storage (no server/dependency) for offline usage  
- Notification or alert (if implemented) to remind you of tasks
